重新学习spring boot

作者: Elara
发表于2025-10-20
更新于2025-10-21

在大二下的时候系统地学习过一段时间的 Spring Boot 和 Spring Cloud。Spring Boot 跟着完成了一个简单的外卖系统。
不过过去时间太长，有些知识点忘了，这几天又重新翻了一下。

一、Spring Boot 的核心思想

可以简单理解为一个大 Map：
key = string, value = object
项目里有很多类，Spring Boot 负责扫描并把它们放进这个容器中。
通过注解 @Component、@Controller、@Service 等让框架识别并注册。

二、常用注解

  • @Controller：控制层，负责对外暴露接口。

  • @Service：业务层，包含具体逻辑。

  • @Mapper：数据访问层（MyBatis 提供）。

    • Controller 调用 Service
    • Service 调用 Mapper 完成数据库 CRUD。

  • @Autowired：自动注入，Spring 会为你找到匹配的 Bean。

  • @Configuration：标记配置类，等价于 XML 配置。

  • @Bean：写在配置类的方法上，把返回对象注册为 Bean（常用于引入第三方组件）。

三、Web 常见注解

  • @RestController：@Controller + @ResponseBody，返回 JSON。
  • @RequestMapping：定义路由，可以指定 URL 与请求方法。
    • @GetMapping
    • @PostMapping
    • @PutMapping
    • @DeleteMapping
  • @RequestParam：获取请求参数。
  • @PathVariable：获取 URL 中的路径变量。

以上内容算是一次快速复习，后面还会继续深入了解。
